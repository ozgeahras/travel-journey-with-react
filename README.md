## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/travel-journey-with-react.git
```

Go to the project directory

```bash
  cd ..travel-journey-with-react
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run dev
```
